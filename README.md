# submarine-war

# 对象索引：

1. 角色
   1. 我方舰船
   2. 敌方舰艇
   3. 敌方指挥舰艇（可随机掉落道具）
2. 角色属性
   1. 移动速度
   2. 血量
   3. 弹药库
3. 道具
   1. hp回复：一次回复玩家最大生命值的30%
   2. 水雷爆炸范围提升：一次提升水雷爆炸范围30%
4. 玩家技能
   1. 超级武器
5. 玩家记分

# loading

显示加载进度数值

# 首页

1. 有一个技能配置面板，玩家一共5个技能点，可自由分配：
   1. 移动速度：满级3点，影响移动速度
      1. 船的外观加装推进器
   2. 血量：满级3点，影响玩家初始血量
      1. 外观增加更多钢板，变成铁甲船
   3. 弹药库：满级2点，玩家初始有3个水雷库存，影响玩家库存水雷数量
2. 技能点分配完毕，下方的开始游戏btn亮起，即可进入游戏

## 游戏界面

1. 沉浸式的座舱内包框
   1. hp：玩家被潜艇攻击到-20%，0%则游戏结束
   2. mp：消灭一个地方潜艇能量+10%，100%即可使用超级武器
   3. 水雷数量显示

1) 玩家控制舰船在海平面左右移动，方向键控制，空格键向下投递水雷
   1. 玩家的hp和mp显示在画面左上角
   2. 玩家舰船的水雷库存数量显示在画面右上角
   3. 超级武器：玩家mp满后可以按z键开启超级武器，当前屏幕内的潜艇时停0.5秒，播放动画全部被消灭清屏
2) 敌方潜艇从右向左移动，随机发射导弹攻击玩家
   1. 导弹的路径：导弹离开潜艇后，会随机向左或右斜上方45度前进，到达一定高度后转为向上垂直攻击
   2. 碰到玩家的水雷会发生爆炸
   3. 爆炸后变成碎片，缓慢向下沉入海底，逐渐消失
   4. 1个潜艇爆炸时会发生连锁爆炸
3) 敌方指挥舰艇
   1. 攻击频次和节奏和常规潜艇一致，玩家击败后会随机提供道具，道具向上飘浮供玩家拾取

# 游戏失败

1. 重新挑战：以当前战船配置再次挑战
2. 重新配置加点：回到初始加点界面

# 游戏节奏

前10个潜艇为适应训练，让玩家掌握各项操作，逐渐加大潜艇数量密度和导弹攻击速度，直到玩家hp归零

# 游戏乐趣

通过玩家初试配置技能点，差异化配置自己的舰船性能，挑战排行榜最高分
